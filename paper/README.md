# HSCI V4 — Research & Specification Directory (paper/README.md)

This directory serves as the authoritative home for the scientific research papers, execution proofs, and experimental evaluations documenting the Hyper-Symbolic Cognitive Invention (HSCI) v4 architecture.

---

## 1. Directory Structure

*   `HSCI_V4_Technical_Paper.md`: The primary research paper containing logical specifications, comparisons, and Mermaid diagrams.
*   `figures/`: Diagram exports and pipeline files.
*   `tables/`: Performance matrices.
*   `references/`: Bibliographic references.
*   `experiments/`: Evaluation run benchmarks logs.
*   `appendix/`: Z3 math assertions detail and schema files.
*   `drafts/`: Notes for subsequent releases.

---

## 2. Living Document Rules

To maintain high scientific rigor:
1.  **Synchronized updates**: Any modifications made to core subsystems (e.g. BrainKernel pipeline, WorkingMemory scopes) must update the corresponding sections of `HSCI_V4_Technical_Paper.md`.
2.  **No fabricated claims**: All accuracy and latency metrics must be derived directly from the automated evaluation runner framework execution output.
3.  **Terminology Alignment**: Maintain consistent terminology between implementation classes and research naming conventions.
