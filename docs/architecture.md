# Cognitive Anatomy

HSCI is divided into specialized functional lobes, each responsible for a distinct phase of the deliberation cycle.

## 1. Perception Lobe
Absorbs raw human stimulus and identifies actors (entities) and relationships (constraints). It uses the **Symbolic Knowledge Graph** to normalize language.

## 2. Logic Lobe
The gateway to formalization. It constructs the **Universal Symbolic Specification (Σ)**, a mathematical contract representing human intent.

## 3. Verified Lobe
The mathematical heart of the system. It utilizes the **Microsoft Z3 SMT Prover** to guarantee that every hypothesis is logically consistent and proven correct.

## 4. Memory Lobe
The episodic storage unit. It build "Symbolic Intuition" by recording and retrieving past solving successes.
