# Welcome to HSCI

The **Hyper-Symbolic Cognitive Invention (HSCI)** is a fundamental shift from statistical prediction to axiomatic cognitive reasoning.

## Mission
Our goal is to build an autonomous reasoning engine that is:
- **Deterministic**: Zero hallucination by design.
- **Explainable**: Every thought provides a transparent cognitive trace.
- **Self-Teaching**: Learns new concepts from single lessons via Instructional Priming.

## Quick Start
To launch the core engine:
```bash
python run_app.py
```
Then visit `http://localhost:8000` to interact with the brain.
